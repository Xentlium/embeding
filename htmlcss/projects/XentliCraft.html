XentliCraft – A Mod for Minecraft

XentliCraft is my first attempt to make a mod for Minecraft, and it turned out pretty nice if you ask me.

It's a pretty basic mod, but I like the features (don't remind me the Bacon is a bit OP; I already know it). The mod adds bacon, compressed cobblestone and some tools. Everything is mentioned below.

Items and blocks in the mod

  • - Bacon
  • - Bacon ore
  • - Four kinds of Compressed Cobblestone
  • - One kind of ingots and tools for every type of cobble (Mentioned in table below)

Download

You can download the latest version (0.6) here.
It recurires Minecraft Forge 9.11.1.965 and Minecraft 1.6.4. NOTE: I'll update the mod to 1.7.4 when I get time to do so.
